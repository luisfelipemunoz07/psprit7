# psprit7   
## Analisis de datos de autos en venta
Esta aplicacion permite a los usuarios explorar los datos recolectados por un consecionario de autos, para entender el comportamiento de los datos.
## Esta aplicación web permite a los usuarios:
- Visualizar histogramas del kilometraje (odómetro) de los vehículos
- Crear gráficos de dispersión para analizar la relación entre el kilometraje y el precio
- Interactuar con los datos mediante botones o casillas de verificación
- Explorar patrones en los datos de venta de coches
## Cómo usar la aplicación
1. Ejecuta la aplicación con `streamlit run app.py`
2. Usa los controles interactivos para generar diferentes visualizaciones
3. Explora los datos mediante los gráficos generados

## Tecnologías utilizadas
- Python
- Streamlit
- Plotly Express
- Pandas
